# Opis projektu
Projekt dotyczy aplikacji wspomagającej studentów w organizacji nauki i zarządzaniu zadaniami.

## Opis problemu
Rozwiązuje problem braku planu nauki, nadmiaru informacji i trudności z priorytetyzacją zadań.

Głównymi odbiorcami są studenci w wieku 18–25 lat, którzy chcą efektywniej planować czas i śledzić postępy. 

---

Funkcje obejmują:
- listę zadań,
- kalendarz tygodniowy,
- przypomnienia
- oraz proste statystyki postępów.
